Portafolio Personal

Descripción

El sitio contiene varias secciones fundamentales para mostrar un portafolio de desarrollador o creativo:

Header: incluye el logo, título y enlaces sociales.
Menú de navegación: enlaces a las secciones principales, Acerca de mí, Mis proyectos, Contacto.
Sección Acerca de mí: breve presentación personal.
Sección de proyectos: tarjetas visuales que muestran los proyectos del portafolio.
Seccion de contacto, correo y Github
Footer: pie de página con información final o derechos reservados.
Estilos y diseño

Se usó una paleta de colores oscuros, con blanco y tonos cafés para lograr buen contraste.
Diseño centrado, con uso de Flexbox y Grid.
Transiciones suaves y elementos adaptables.
Diseño responsivo

Se implementaron media queries para asegurar una buena experiencia en dispositivos:

max-width: 480px: móviles
max-width: 768px: tablets
max-width: 1024px: pc
Estructura de clases (CSS) con la metodologia BEM

EJEMPLO

.header__titulo
.contenedor__titulo
.menu__ul
.contenedor__tarjeta img
Esto facilita mantener y escalar los estilos.

Cómo usar

Clonar este repositorio.
Abrir el archivo index.html en tu navegador.
